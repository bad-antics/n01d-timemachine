# N01D Time Machine Wiki

Welcome to **Time Machine** — an authentic classic computing experience with unified emulator launching.

## Supported Systems

| System | Emulator | Status |
|--------|----------|--------|
| Commodore 64 | VICE | ✅ |
| Amiga | FS-UAE | ✅ |
| ZX Spectrum | Fuse | ✅ |
| Apple II | LinApple | ✅ |
| Atari ST | Hatari | ✅ |
| DOS | DOSBox | ✅ |
| NES | Nestopia | ✅ |
| SNES | Snes9x | ✅ |

## Quick Start

```bash
git clone https://github.com/bad-antics/n01d-timemachine
cd n01d-timemachine
cargo build --release
./target/release/n01d-timemachine
```
